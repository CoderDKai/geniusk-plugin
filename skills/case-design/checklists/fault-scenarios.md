# 故障场景 Checklist

生成故障场景用例时，对照以下清单检查是否遗漏。

## 网络相关
- [ ] 请求超时
- [ ] 网络断开/恢复
- [ ] DNS 解析失败
- [ ] SSL/TLS 证书错误
- [ ] 响应数据不完整

## 缓存相关
- [ ] 缓存过期
- [ ] 缓存与数据库不一致
- [ ] 缓存击穿（热点 key 失效）
- [ ] 缓存雪崩（大量 key 同时失效）
- [ ] 缓存穿透（查询不存在的数据）

## 并发相关
- [ ] 竞态条件（多个请求同时修改同一资源）
- [ ] 死锁
- [ ] 重复提交
- [ ] 乐观锁冲突

## 存储相关
- [ ] 磁盘空间不足
- [ ] 数据库连接池耗尽
- [ ] 文件读写权限不足
- [ ] 数据库查询超时

## 服务依赖
- [ ] 第三方服务不可用
- [ ] 第三方服务响应异常（格式错误、状态码异常）
- [ ] 第三方服务限流
- [ ] 消息队列积压/丢失

## 资源限制
- [ ] 内存不足
- [ ] CPU 占用过高
- [ ] 文件句柄耗尽
- [ ] 请求体/响应体超过大小限制
