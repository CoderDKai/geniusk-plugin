# 用户流程 Checklist

生成用户流程用例时，对照以下清单检查是否遗漏。

## 中断操作
- [ ] 中途返回上一步
- [ ] 中途取消整个流程
- [ ] 关闭页面后重新打开（是否恢复进度）
- [ ] 浏览器前进/后退按钮

## 重复操作
- [ ] 重复提交表单
- [ ] 重复点击按钮
- [ ] 刷新页面后重复提交（POST 重放）

## 异常路径
- [ ] 跳过必要步骤直接访问后续页面
- [ ] 通过 URL 直接访问中间步骤
- [ ] 在流程中切换账号
- [ ] 在流程中权限被撤销

## 多入口
- [ ] 从不同入口进入同一流程（如从列表页 vs 从通知）
- [ ] 深链接/书签直接进入流程中间步骤
- [ ] 从邮件/推送通知跳转进入
