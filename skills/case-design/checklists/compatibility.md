# 兼容性 Checklist

生成兼容性用例时，对照以下清单检查是否遗漏。

## 浏览器
- [ ] Chrome（最新两个大版本）
- [ ] Firefox（最新两个大版本）
- [ ] Safari（最新两个大版本）
- [ ] Edge（最新两个大版本）
- [ ] 移动端浏览器（Safari iOS、Chrome Android）

## 设备/屏幕
- [ ] 桌面端（1920×1080、1366×768）
- [ ] 平板（768×1024）
- [ ] 手机（375×667、390×844）
- [ ] 超宽屏（2560×1440）
- [ ] 横屏/竖屏切换

## 操作系统
- [ ] macOS
- [ ] Windows
- [ ] Linux（如适用）
- [ ] iOS
- [ ] Android

## 国际化
- [ ] 不同语言的文本长度导致布局变化
- [ ] RTL（从右到左）语言
- [ ] 不同时区
- [ ] 不同日期/数字格式
- [ ] 多字节字符（中日韩）
