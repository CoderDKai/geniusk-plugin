# 可访问性（A11Y）Checklist

生成可访问性用例时，对照以下清单检查是否遗漏。

## 键盘操作
- [ ] 所有交互元素可通过 Tab 键聚焦
- [ ] 焦点顺序合理
- [ ] 可通过 Enter/Space 触发操作
- [ ] 可通过 Escape 关闭弹窗/下拉
- [ ] 焦点陷阱（模态框内焦点不外泄）

## 屏幕阅读器
- [ ] 所有图片有 alt 文本
- [ ] 表单元素有关联的 label
- [ ] 动态内容变化有 aria-live 通知
- [ ] 自定义组件有正确的 ARIA 角色
- [ ] 错误提示能被屏幕阅读器读取

## 视觉
- [ ] 颜色对比度达标（WCAG AA 4.5:1）
- [ ] 不仅靠颜色传达信息
- [ ] 文字可放大至 200% 不破坏布局
- [ ] 动画可暂停/关闭（prefers-reduced-motion）
