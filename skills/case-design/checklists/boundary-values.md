# 边界值 Checklist

生成边界值用例时，对照以下清单检查是否遗漏。

## 数值类型
- [ ] 最小值
- [ ] 最小值 - 1
- [ ] 最大值
- [ ] 最大值 + 1
- [ ] 零
- [ ] 负数
- [ ] 小数/浮点精度（0.1 + 0.2）
- [ ] 极大数（溢出）

## 字符串类型
- [ ] 空字符串
- [ ] 单个字符
- [ ] 最大长度
- [ ] 超过最大长度
- [ ] 纯空格/制表符
- [ ] 特殊字符（<>&"'/\）
- [ ] Unicode / Emoji
- [ ] SQL 注入字符（' OR 1=1 --）
- [ ] XSS 字符（<script>）
- [ ] 前后带空格

## 集合/列表类型
- [ ] 空集合
- [ ] 单个元素
- [ ] 最大数量
- [ ] 超过最大数量
- [ ] 包含重复元素

## 日期/时间类型
- [ ] 最早日期
- [ ] 最晚日期
- [ ] 跨时区
- [ ] 夏令时切换
- [ ] 闰年 2 月 29 日
- [ ] 时间戳为 0（1970-01-01）
- [ ] 未来日期

## 文件类型
- [ ] 空文件（0 字节）
- [ ] 极大文件
- [ ] 不支持的格式
- [ ] 文件名含特殊字符
- [ ] 文件名超长
- [ ] 损坏的文件
